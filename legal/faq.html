두 사람이 같은 기기로 해도 되나요?▼
네, 가능해요. 한 기기에서 두 사람이 번갈아 선택하면 돼요. 각 문항마다 나(A)와 파트너(B) 둘 다 선택해야 다음으로 넘어가는 구조예요.
각자 따로 해도 되나요?▼
현재는 두 사람이 함께 같은 화면에서 진행하는 방식이에요. 각자 따로 하는 기능은 준비 중이에요.
MBTI와 다른가요?▼
42world는 MBTI를 기반으로 하되, 커플 관계에 특화된 8가지 유형으로 재설계했어요. MBTI 유형과 대응되지만 연애·갈등·소통에 초점을 맞춘 분석이에요.
결과 페이지가 안 나와요.▼
두 사람 모두 24문항을 완료해야 결과가 나와요. 한 사람이라도 미선택 문항이 있으면 다음으로 넘어가지 않아요. 처음부터 다시 시도해보시고, 그래도 안 되면 아래 이메일로 문의해 주세요.
리포트는 어떻게 구매하나요?▼
검사 결과 페이지 하단의 "리포트 구매하기" 버튼을 누르면 결제 페이지(래피드)로 이동해요. 결제 완료 후 다운로드 페이지로 자동 이동돼요.
결제 방법은 뭐가 있나요?▼
카드 결제, 카카오페이, 네이버페이 등 래피드에서 지원하는 모든 결제 수단을 사용할 수 있어요.
얼리버드 가격은 언제까지인가요?▼
얼리버드 특가(9,900원)는 초기 한정 수량으로 운영되며, 예고 없이 종료될 수 있어요. 현재 구매하시는 게 가장 저렴해요.
결제했는데 다운로드 페이지로 안 넘어가요.▼
결제 완료 후 자동으로 다운로드 페이지로 이동해요. 자동 이동이 안 됐다면 아래 버튼을 직접 눌러 접속해 주세요.
